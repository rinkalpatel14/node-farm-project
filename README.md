# Node Farm Project

A Node.js + Express + EJS based web application that displays product data from a JSON file.

## Features
- Product overview page
- Product detail page
- Dynamic routing
- Data loaded from JSON file

## Technologies Used
- Node.js
- Express.js
- EJS
- JSON

## Installation

1. Clone the repository
git clone https://github.com/rinkalpatel14/node-farm-project.git

2. Install dependencies
npm install

3. Start the server
nodemon app.js

4. Open in browser
http://localhost:3000
